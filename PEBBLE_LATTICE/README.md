# PEBBLE LATTICE™

**"Below the Seed"**

## Overview

The PEBBLE LATTICE™ is a foundational identification system for the NEXUS_NAIR architecture. This directory contains the pebble IDs for all 13,713 verified brands.

## Structure

Each brand receives a unique Pebble ID in the format:
```
PBL-[HASH8]-[BRANDID5]
```

Example: `PBL-A3F7B2C1-00542`

### Components

- **PBL** - Pebble Lattice identifier prefix
- **HASH8** - 8-character MD5 hash (uppercase) derived from brand name and ID
- **BRANDID5** - 5-digit zero-padded brand ID

## Features

- **Cryptographic Verification**: Each pebble ID is generated using the Gorilla Codex™
- **9s PulseGlow™**: Synchronized every 9 seconds
- **Immutable**: Once assigned, pebble IDs are permanent
- **R2 Backed**: Mirrored to Cloudflare R2 for redundancy
- **Offline Ready**: Compatible with Offline VaultNode™ (2026)
